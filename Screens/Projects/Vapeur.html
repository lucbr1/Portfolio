En bref

Application full stack (Node.js et Handlebars) représentant une bibliothèque de jeux vidéo.

Backend : Node.js et Express.js pour la gestion des requêtes et la communication avec la base de données.

Frontend : Handlebars pour le rendu des pages HTML en templates.

Projet mené en groupe sur 30h dans le cadre d'un projet de web du BUT informatique graphique.

Contexte

Ce projet a été effectué dans le cadre d'un projet de web du troisième semestre de BUT informatique graphique en groupe de trois.

Nous devions réaliser une application web full stack en utilisant les langages HTML, CSS, JavaScript. Nous devions utiliser Node.js, Express.js, Prisma avec SQLite et Handlebars.

Cette application permet à l'utilisateur de gérer sa bibliothèque de jeux vidéo en utilisant les méthodes CRUD (Créer, Lire, Mettre à jour, Supprimer).

Il a représenté environ 30h de travail.

  • Intégrer la base de données à l'application.
  • Gérer le routage des requêtes.

  • Mettre en place les méthodes CRUD.

Conclusion

Ce projet a été une étape déterminante dans ma compréhension du développement Full Stack. La mise en place d'un système CRUD complet m'a permis de lier concrètement la logique serveur (Node.js) à la persistance des données avec Prisma.

La principale difficulté a résidé dans la gestion des relations entre les jeux, les genres et les éditeurs, ce qui a renforcé ma rigueur dans la structuration des bases de données.

Cette expérience m'a apporté une vision globale d'une application web, de la base de données jusqu'au rendu dynamique avec Handlebars.
